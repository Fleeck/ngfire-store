# Firebase & AngularJS store

This app is using [Firebase](https://firebase.google.com/) and [AngularJS](https://angularjs.org/)

## Prerequisites:
  * Node & npm - https://nodejs.org/
  * Bower - https://bower.io/
  
## Running a local server:
  * `npm install`
  * `bower install`
  * `grunt serve`
